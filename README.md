## 🚀 **Getting Started**

### Prerequisites
- Node.js 16+ installed
- MetaMask or compatible Web3 wallet
- Supported blockchain network tokens
- MongoDB (for backend features)

### Frontend Installation
```bash
npm install
npm start
```

### Backend Installation (Essential Features)
```bash
cd backend
npm install
npm run dev
```

The backend provides critical services:
- **Gas Price Optimization** - Save 20-30% on fees
- **Address Validation** - Prevent scam addresses
- **Transaction Monitoring** - Real-time status updates
- **CSV Processing** - Server-side validation
- **Analytics** - Transaction history and insights

See `backend/README.md` for detailed setup instructions.

### Usage
1. **Connect Wallet**: Click "Wallet Connect" 
2. **Select Network**: Choose your blockchain (BSC/Polygon/Avalanche)
3. **Enter Token**: Contract address or select native currency
4. **Add Recipients**: Upload CSV or enter manually
5. **Send**: Approve and execute bulk transfer
